The lwIP driver for the Stellaris LM3S MCUs


arch/bpstruct.h    - Used to define beginning of packed structures.
arch/cc.h          - System defines for Stellaris/DriverLib conventions.
arch/epstruct.h    - Used to define end of packed structures.
arch/perf.h        - Stub performance monitor include file (not used).

netif/eth_driver.h - public C prototypes for Ethernet driver interface
netif/eth_driver.c - lwIP C Ethernet driver for Luninary Micro Stellars devices

netif-cpp/eth_driver.hpp - public C++ prototypes for Ethernet driver interface
netif-cpp/eth_driver.cpp - lwIP C++ Ethernet driver for Luninary Micro Stellars devices
